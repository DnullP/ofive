# ofive Browser E2E

Browser E2E 验证用户路径、工作台交互、面板组合、tab 生命周期和浏览器层行为。

## 维护要点

1. 用户可见流程变化应补 E2E。
2. 工作台布局、tab 和 panel 生命周期属于 E2E 高价值区域。
3. Browser E2E 不等同于桌面宿主验证。
