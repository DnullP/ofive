# ofive Feature Owner

Feature Owner 是某个能力域的主要责任归属。它用于变更前判断谁拥有行为、谁负责验证、谁需要同步文档。

## 维护要点

1. 用户界面入口归插件或宿主组件。
2. 本地内容事实归 Vault。
3. 派生查询归查询索引或语义索引。
4. AI 工具归 capability owner。
5. 跨组件状态归 Managed Store。
